# Embedded Systems Career Guide

> A comprehensive Android learning application for embedded systems career development with AI-powered features.

## 🎯 Overview

This app provides a gamified learning experience for students and professionals pursuing careers in embedded systems. It features:

- **16-Stage Learning Path** - Structured curriculum from basics to advanced topics
- **AI-Powered Quizzes** - Dynamic questions generated by Gemini API
- **Personalized Assessment** - Initial skill evaluation with detailed reports
- **AI Chat Tutor** - Interactive learning assistant
- **Cloud Sync** - Firebase-based progress across devices

## 🚀 Quick Start

1. Clone the repository
2. Open in Android Studio
3. Add `GEMINI_API_KEY` to `local.properties`
4. Configure Firebase with `google-services.json`
5. Build and run
